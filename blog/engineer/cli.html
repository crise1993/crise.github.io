# 从 0 打造属于自己的脚手架

该文主要介绍如何搭建一个企业自己的项目脚手架

# 搭建前准备

兼容性注意: 需要 Node.js 版本 >= 16.0.0。

  1. 第一步: 全局安装wzh-cli依赖包
npm install wzh-cli -g
  1. 第二步: 在需要创建项目文件目录下打开 cmd 运行以下命令
wzh create xxx
